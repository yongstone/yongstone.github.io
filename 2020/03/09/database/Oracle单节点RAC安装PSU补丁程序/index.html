Oracle 单节点RAC安装PSU补丁程序

描述：单节点RAC，数据库版本12.2.0.1

下载最新OPatch，补丁编号6880880,升级Grid和OracleHome的OPatch

补丁冲突检测（视情况而定，需要配置OPath的环境变量）

系统空间检测（视情况而定）

设置root用户环境变量，使用opathcauto来自动安装补丁集

安装补丁，有bug，文档编号：2336186.1

-nonrolling 模式安装补丁

查看补丁信息

注：如果在安装的补丁的时候数据库已经建好了，在给Grid安装补丁的时候，会把数据库的补丁一起安装上，如果只是安装了Grid和数据库软件就开始安装补丁的话，数据库需要单独安装补丁。给数据库安装补丁需要先关闭数据库，如果出现Prerequisite check “CheckActiveFilesAndExecutables” failed需要关闭数据库以及清理进程。

设置oracle用户环境变量，给数据库安装补丁

开启数据库，运行datapatch

编译无效对象
